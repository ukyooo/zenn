Python が使える環境で `csvkit` をインストールする。

```bash
brew install csvkit
```
or
```bash
pip install csvkit
```

`csvformat` というコマンドが使えるようになるので下記の要領で実行する。

```bash
csvformat -T input.csv > output.tsv
```
